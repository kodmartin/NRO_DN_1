# NRO_DN_2
Repozitorij za drugo domačo nalogo
V izvedenih meritvah se je kot najhitrejša izkazala ročno implementirana bilinearna interpolacija, saj neposredno poišče celico, ki vsebuje iskano točko, in nato izvede le nekaj 
osnovnih aritmetičnih operacij. Metoda griddedInterpolant je sicer zelo hitra, vendar še vedno zahteva pripravo celotne mreže in pretvorbo podatkov v ustrezno obliko. 
Najpočasnejša je metoda scatteredInterpolant, ker deluje na razpršenih podatkih in mora zato interno poiskati ustrezne točke ter izvesti zahtevnejše postopke interpolacije.
